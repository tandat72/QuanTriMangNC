Protect the ISA Server Computer

You have successfully installed Microsoft Internet Security and Acceleration (ISA) Server 2006 on your computer. Note the following security recommendations:

  • Turn on Microsoft Update so that you will be notified when the latest updates are available. When you turn on Automatic Updates, the Microsoft Update Web site is regularly checked for high-priority updates that can help protect your computer. These updates can include security updates, critical updates, and service packs. Depending on the setting you choose, ISA Server automatically downloads and installs any high-priority updates that your computer needs, or notifies you as these updates become available. We recommended that you configure Automatic Updates to download updates automatically, but to allow you to choose when to install them. For instructions on configuring Automatic Updates, see the Microsoft Update Web site.
  • For more detailed information on securing the computer running ISA Server, including information on the Security Configuration Wizard (provided with Microsoft Windows Server™ 2003 with Service Pack 1), see the ISA Server Security Guide.
  • Use the Microsoft ISA Server Best Practices Analyzer Tool to determine the overall health of your ISA Server computers and to diagnose current problems. The tool scans the configuration settings of the local ISA Server computer and reports issues that do not conform to the recommended best practices. Download the tool from Microsoft Download Center.

To view this page again after closing it, do one of the following:

  • Open the Secure.htm file, located in the ISA Server installation directory on the ISA Server computer.
  • Save this file to a shared folder on your network, which you can access as necessary.
