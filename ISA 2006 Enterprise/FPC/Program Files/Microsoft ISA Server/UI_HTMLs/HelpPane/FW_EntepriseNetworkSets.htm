Network Sets
After configuring enterprise networks, you can group them into enterprise network sets, which can be used in enterprise and array-level rules.

There are two predefined network sets: All Networks, which excludes all networks that are unknown to ISA Server, and All Protected Networks, which excludes the unprotected External network.
