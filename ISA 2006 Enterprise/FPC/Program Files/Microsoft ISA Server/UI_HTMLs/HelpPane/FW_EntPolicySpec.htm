Enterprise Policy
After you create an enterprise policy, you create access rules for the enterprise policy. Enterprise policy access rules can be ordered before the array policy is processed.

A default rule which denies all access, is included in every enterprise policy and cannot be deleted or modified. The default policy (named Default Policy) includes only this default rule.
