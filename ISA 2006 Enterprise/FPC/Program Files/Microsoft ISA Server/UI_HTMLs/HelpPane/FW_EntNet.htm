Enterprise Networks
An enterprise network is composed of IP ranges, and it is global to all the arrays in the enterprise. Enterprise networks do not have any of the other properties that you would define for array-level networks.

Enterprise administrators can create and modify enterprise networks. Array administrators can view enterprise networks and include them in an array-level network.
