Networks and Network Templates
You can configure the networking environment by selecting an ISA Server network template, or by defining networks and the relationships between them.

Selecting an ISA Server network template runs the Network Template Wizard. Based on the template, the wizard creates networks, network rules, and firewall policy rules. In addition, system policy rules are enabled to allow access to required services.
