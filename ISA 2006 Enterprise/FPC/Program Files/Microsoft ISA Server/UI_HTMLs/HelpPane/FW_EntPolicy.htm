Enterprise Policies
An enterprise policy includes a set of access rules. The policy can be applied to arrays in the enterprise and can be augmented by the array's own policy.

Enterprise administrators can create enterprise policies, and configure and view existing enterprise policies. Array administrators can view existing enterprise policies.
