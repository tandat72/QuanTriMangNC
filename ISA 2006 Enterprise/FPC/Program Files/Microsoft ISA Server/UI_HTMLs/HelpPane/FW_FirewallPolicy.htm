Firewall Policy
You can secure your network assets by defining firewall policy rules allowing or denying access to your connected networks, Web sites, and servers. Use the rule elements, available in the Toolbox, to specify rule conditions.

When you install ISA Server, a set of system policy rules for the ISA Server computer is configured by default.
